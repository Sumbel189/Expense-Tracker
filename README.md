# Expense Tracker - Трекер расходов

**Автор:** Фатыхова Сюмбель

## Описание программы

Expense Tracker — это десктопное приложение для отслеживания личных финансов. Программа позволяет:

- Добавлять расходы с указанием суммы, категории и даты
- Просматривать все расходы в табличном виде
- Фильтровать расходы по категории и временному периоду
- Подсчитывать итоговую сумму за выбранный период
- Сохранять и загружать данные в формате JSON
- Удалять ненужные записи
- Валидация вводимых данных (сумма > 0, корректный формат даты)

## Требования

- Python 3.7 или выше
- Стандартные библиотеки: tkinter, json, datetime (входят в Python)
